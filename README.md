nand2tetris
===========

Repo my work on the book _The Elements of Computing Systems_, aka.
*nand2tetris*.


# Setup


```
git clone git@github.com:trammell/nand2tetris.git
wget -O /tmp/n2t.$$ https://drive.google.com/file/d/0B9c0BdDJz6XpZUh3X2dPR1o0MUE/view


```
